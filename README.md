# 我的第一个Android APP —— MyMemos
 #### 说明
 大三课设作品，一款风格简约的备忘录, 有时间会继续完善。
 #### 功能
 

 1. 新增备忘录，可设置完成状态，以及分类（暂时只有工作、生活、学习标签），支持富文本编辑器编辑。
 2. 长按删除备忘录。
 3. 可按日期查询以及标签查询，可重新编辑。
